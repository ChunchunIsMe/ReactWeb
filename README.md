## 启动
安装包: `npm i`

运行: `npm run dev`

## mock
1. mock中间件: `config/mock.js`
2. mock数据: mock文件夹下的任意json

2. 封装组件：src/Component/SortableContainer.js  src/Component/SortableItem.js

## 拖动右下角改变元素大小和字体大小
1. 封装组件：src/Component/SortableItem.js

## 懒加载Dom
